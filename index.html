Configuration Firebase
Firebase permet de synchroniser vos données entre tous vos appareils en temps réel — gratuitement. Suivez ces étapes une seule fois :
Étape 1 — Allez sur console.firebase.google.com et connectez-vous avec votre compte Google.
Étape 2 — Cliquez Ajouter un projet, donnez-lui un nom (ex: mon-organisateur), désactivez Google Analytics si proposé, puis créez.
Étape 3 — Dans le menu à gauche, allez dans Realtime Database → Créer une base de données → choisissez europe-west1 → mode test.
Étape 4 — Revenez à l'accueil du projet, cliquez sur l'icône </> Web, donnez un nom à l'app, puis copiez les valeurs firebaseConfig dans les champs ci-dessous.
